1.5 Basic Operators
Java provides a rich set of operators to manipulate
variables. We can divide all the Java operators into the following
groups:
1. Arithmetic Operators
2. Relational Operators
3. Bitwise Operators
4. Logical Operators
5. Assignment Operators
6. Miscellaneous Operators

 Logical Operators:
Logical operators are the operators which are used to form
multiple conditions by combining two or more relations.
The following table lists the logical operators:
Operator
Description
&&
And operator. Performs a logical conjunction on
two expressions.(if both expressions evaluate to
True, result is True. If either expression evaluates
to False, result is False)
||
Or operator. Performs a logical disjunction on two
expressions.(if either or both expressions evaluate
to True, result is True)
!
Not operator. Performs logical negation on an
expression.
