9.1 Introduction of I/O file
➢ When variables and arrays are used to store data in program, it
has some problems like data is lost when variable goes out of its
scope or when program terminates.
➢ The above problem can be solved by string data on secondary
storage. Such as floppy disk or hard disk by using concept of
files.
➢ Data stored on files is called permanent or persistent data.
➢ A file is a collection of related records placed in a particular area
on the disk.
➢ A record is composed of several fields and a field as a group of
characters.
➢ Characters in java are Unicode characters composed of two
bytes each byte containing eight binary digits 1 or 0.
➢ Data storing and data management using files is called as file
processing, includes task like creating file, updating files and
manipulation of data.
