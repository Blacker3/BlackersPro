8.2.1 Building applet code
It is essential that user created applet code uses the
services of two classes applet and graphics from the java class
library. An applet code starts with import statements:
1) Import java .applet.*;
This import statement imports applet package, which
contain the class applet.
2) Import java.awt.*;
This import statement imports the Abstract Window Toolkit
(AWT) used for user interraction.
3) Subclass of an applet class must be defined.
For example: Public class hellojava extends Applet
The ‘extends’ keyword specifies hellojava class is a sub
class of the Applet class hellojava class is a sub class of the
super class applet.
4) The paint() method requires Graphics object as an
argument,defined as follows:
Public void paint (Graphics g)
Java awt package continues the graphics class so applet code imports
it.
➢ The applet code have general format like:
Import java .awt.*;
Import java. Applet.*;
…
…
Public class appletclassname extends Applet
{
…
…
Public void paint (Graphics g)
{
…
… // Applet operations code
…
}
…
…
}
Program:
Import java .awt.*;
Import java. applet.*;
Public class hellojava extends Applet
{

{
g.drawString (“Hello Java”, 11,110);
}

8.2.2 Designing a web page
Java applets are programs that reside on a web page.
To run the java applet, the web page that references that applet
is required.
HTML is a Hyper Text Markup Language.
➢ The web page is divided into following three sections:
1) Comment section (optional)
2) Head section (optional)
3) Body section.
1) Comment section(optional):
It holds the comments about web page, comments must be
informative. A comment line begins with ‘<!’And ends with a
‘>’.
2) Head section(optional):
The head section enclosed within starting <Head>tag and
closing </Head>tag.
It is nothing but title of web page shown below:
<Head>
<Title>Welcome</Title>
</Head>
A slash (/) is any tag indicates end of tag.
3) Body section:
It contains entire information about web page and its
behavior. Several tags can be used is body section for
appearance of the web page.
<Body>
…….. //applet tag
</Body>
