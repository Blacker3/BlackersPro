8.6 Methods using in Applet
8.6.1 Update() Method
Update() Method is called when applet request to redraw a
portion of window.
Sometimes method overriding is used with paint () and update
() method because default version of update () will fills an applet
with the default background color and then calls the paint ()
method.
Syntax:
Public void update (){
//redraw a window
}
Public void paint (Graphics g){
Update ();
}
8.6.2 Repaint() Method
When paint () method is called by AWT then only applet
writes to the output window. Whenever applet needs to change or
update the output window then it calls repaint () method defined by
AWT.
In turn the AWT will then execute the paint (), which will
displays the stored information. The repaint () method has
following four forms:
1. Void repaint():
Entire window is repainted.
2. Void repaint(int left, int top, int width, int height):
It specifies the region that will be repainted. The upper
left coordinates are specified in left and top. The width and
height also passed in last two parameters.
3. Void repaint():
Here, maxDelay is used to provide the delay. It is a
maximum no. of milliseconds that can elapse before paint ()
method is called.
4. Void repaint(long maxDelay, int left, int top, int width, int
height):
Here, maxDelay is used to provide the delay. It is a
maximum no. of milliseconds that can elapse before paint ()
method is called. The dimension of portion of the window to
be repainted is given in top, left, width and height parameter in
pixels.
8.6.3 Using control loops in Applet
All control structures can be used in Applet likes for, while,
do...while.
8.6.4 Using Fonts in Applet
A font determines look of the text when it is painted. Font is
used when painting text on a Graphics context and is a property of
AWT component. Fonts are encapsulated in font class. A Font has
three properties that contribute the appearance of the text
1. Family name : specifies general name of the font (e.g.
Courier)
2. Logical name : specifies the category of the font (e.g.
Monospaced)
3. Face name : specifies specific font (e.g. Courier Italic)
➢ To select a new font first create a font object by using
following constructor:
Font(String fontName, int fontStyle, int pointSize)
Where,
fontName specifies name of font.
Java environment support, Dialog, DialogInput, serif, Sans
Serif, Monospaced and Symbol. Dialog is default font used by
applet.
Fontstyle has one of the three constant:Font.PLAIN,
Font.BOLD, Font.ITALIC. To combine style OR them together
like Font.BOLD|Font.ITALIC which specifies bold as well as italic
style.
The pointSize is size of the font in points.
